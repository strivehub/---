# conventional-algorithm-
对于常见的算法的实现和解释                                                                                                         
quick_sort.py 快速排序算法：是利用递归的方法，实现对一数组进行排序的                                                                   
select_sort.py 选择排序算法：利用比较前后两个数，每一次比较就将最小的数字放在前面                                                         
factorial.py 阶乘的计算：利用递归的思想，计算一个数的阶乘
